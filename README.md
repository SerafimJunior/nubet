# nubet
Casa de apostas

## First-version (feature/first-version-nubet)

Este repositório contém uma versão inicial estática do site NUBET.

- Paleta de cores:
	- Primário: roxo
	- Secundário: rosa pink
	- Terciário: laranja fluorescente

- Headline: "Deposite 100 reais e ganhe mais 100"
- Subtítulo: "As melhores Odd's do mercado"
- Aviso: Apostas apenas para maiores de 18 anos.

Arquivos adicionados:

- `index.html` — página principal
- `css/styles.css` — estilos com as cores do projeto
- `assets/background.svg` — fundo em SVG com gradiente

Como ver localmente:

1. Abra um terminal na pasta do repositório.
2. Rode um servidor simples (exemplo com Python 3):

```powershell
python -m http.server 8000
```

3. Abra `http://localhost:8000` no navegador.

Observações: o arquivo `css/styles.css` referencia `assets/background.jpg` caso você adicione uma imagem raster; o SVG funciona como fallback.
